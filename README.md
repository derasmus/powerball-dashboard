# 🎱 SA Powerball Analytics Dashboard

A responsive, dark-themed analytics dashboard for South Africa Powerball lottery data.

## 📁 Files

- `index.html` - Main dashboard (dynamic, reads from JSON)
- `data.json` - Exported lottery data (auto-generated)
- `export-data.js` - Script to generate data.json from database

## 🎯 Features

- **Live Statistics**: Total draws, hot/cold numbers, best Powerball
- **Number Rankings**: Top 15 hot + bottom 15 cold with visual bars
- **Gap Analysis**: Shows how long since each number appeared
- **Last 20 Draws**: Historical results
- **Smart Recommendations**: Hot, cold, and sticky number combinations
- **Dynamic Updates**: Just run export-data.js to update

## 🔄 Update Dashboard Data

When you add new draws to the database:

```bash
cd /Users/dustinerasmus/.openclaw/workspace/powerball-dashboard
node export-data.js
```

This regenerates `data.json` with fresh stats.

Then push to GitHub (if using GitHub deploy):
```bash
git add data.json
git commit -m "Updated data"
git push
```

Vercel auto-redeploys! 🎉

## 📊 Data Format

The dashboard expects `data.json` with this structure:

```json
{
  "metadata": {
    "totalDraws": 806,
    "dateRange": { "earliest": "2018-06-26", "latest": "2026-02-27" },
    "lastUpdated": "2026-03-02T20:01:00Z",
    "lastDraw": { ... }
  },
  "hotNumbers": [...],
  "coldNumbers": [...],
  "powerballs": [...],
  "topPairs": [...],
  "last20Draws": [...],
  "recommendations": {...}
}
```

All auto-generated by `export-data.js`!
